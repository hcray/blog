运算符小结

1、i++,比较简单就是i=i+1的缩写，同理i--就是i=i-1的缩写 
	int i = 2, j = 0;
	j = i++;
	System.out.println("i: " + i + " j: " + j);

结果：
	i: 3 j: 2
2、++i,也很简单，其实也是i=i+1,但是如果j=++i,就不一样了，先对i进行自增，然后在赋值给j;--同理 
	int i = 2, j = 0;
	j = ++i;
	System.out.println("i: " + i + " j: " + j);

结果：
	i: 3 j: 3
3、+= 就是两个数值相加的简写;y+=x即y=y+x; -=,*=,/=同理 
	int i = 2, j = 3;
	j += i;
	System.out.println("i: " + i + " j: " + j);

结果：
	i: 2 j: 5
4、=+这个就比较奇怪了，到底是先加了，还是先赋值呢。j=+i;其实就是赋值语句 
		int i = 2, j = 3;
		j =+ i;
		System.out.println("i: " + i + " j: " + j);

结果：
	i: 2 j: 2
有些人就会说了，那和j=i的结果是一致的，有什么区别呢。程序猿就要刨根问底，于是乎，查看字节码，如下: 可以看出来字节码是一致，而且没有iadd操作，说明没有什么区别，这样写法也会是语法兼容的结果吧。 
		int i = 2, j = 3;
		j = i;
